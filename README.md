# Bitcoin Dashboard

Production-grade Bitcoin analytics dashboard with a Python backend, background workers and a server-driven frontend.

## Overview

This project powers a live Bitcoin analytics platform with:
- multi-process backend workers
- Redis-based caching and synchronization
- direct blockchain and ElectrumX integration
- Flask-served frontend (HTML, CSS, JavaScript)

The repository mirrors the real production structure.  
Configuration and secrets are provided via environment variables and are not part of the repository.

## Status

Actively used in production and continuously evolving.
